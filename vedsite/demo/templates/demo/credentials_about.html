How to get Stepik Client ID and Stepik Client Secret?

Stepik Client ID and Stepik Client Secret are special values that the Stepik Platform uses in order to authenticate applications like this one. In order to be able to utilize the Stepik related features of the app you’ll need to request them first.

How to get the Keys

  1. Log in to a Stepik website and then follow this link: https://stepik.org/oauth2/applications/register/
  2. Once there you’ll need to specify a specific name for your application (it doesn’t really matter what does this field say for the purposes of using this application).
  3. For the Client Type, select Confidential.
  4. For the Client Type, select client-credentials.
  5. Press Save.
  6. Once complete, copy the Client ID and the Client Secret fields into the form on the Credentials Page.
